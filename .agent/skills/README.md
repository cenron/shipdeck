# Project Skills

This folder contains the default skill bundle copied from `Brain/Resources/Project Skills/`.

## Default skills

- `guided-coding`
- `writing-plans`
- `systematic-debugging`
- `verification-before-completion`
- `requesting-code-review`

## Use

- Load the matching skill before answering when the task fits one of them.
- Use `guided-coding` by default for pair-programming help.
