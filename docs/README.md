# Shipdeck Docs

## Overview

- `vision.md` - product intent, MVP boundaries, and current frontier
- `working-mode.md` - implementation collaboration contract and pairing defaults
- `architecture.md` - current system shape, boundaries, and MVP scope
- `implementation-plan.md` - step-by-step build plan for the MVP

- Design and product decisions live in the vault project note for Shipdeck.
- Code and repo-level docs live here in the workspace.
- Implementation session behavior in this repo follows `working-mode.md`.
